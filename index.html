CAD FlexiView

In this project, I aim to increase the user experience by visualizing AutoCAD files. The development process is ongoing and I have the ability to save the AutoCAD 2000 version in AutoCAD 2010; I can also perform the opposite operations. I plan to provide similar functionality for other versions. My project is developed using the C# programming language and is structured using classes based on object-oriented programming principles. I support users' personalized experiences by working with ini files for user settings. It allows users to work more effectively by visualizing according to layer information. I can customize the visualization by changing the colors in the layer settings section. In the future, I aim to further develop and improve the system by adding new functions. This project offers an innovative solution that will help users manage AutoCAD files more effectively.
